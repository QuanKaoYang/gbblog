全角モードのまま半角スペースを入力

電氣羊 @ Goldenbridge   in Osaka
2020年01月11日
  • 小ワザ
  • Word
  • 効率化

日本語入力モード（全角）にしていると、スペースはすべて全角になります。 ところが、文章を書いていると、半角スペースを使いたいときも多いのではないでしょうか。 そのたびに全角／半角を切り替えるのは煩わしい…… そんなときは　Shiftキーを押しながらスペースを押しましょう。 これだけで全角モードでも半角スペースが打ち込めます。
